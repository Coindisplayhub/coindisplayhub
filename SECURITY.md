# Security Policy

## Reporting a Vulnerability

COINDISPLAYHUB takes the security of our platform and user data seriously. If you discover a security vulnerability, we appreciate your help in disclosing it responsibly.

### How to Report

Send vulnerability reports to **security@coindisplayhub.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Your contact information for follow-up

### What to Expect

- **Acknowledgment** within 48 hours of your report
- **Assessment** within 5 business days
- **Resolution timeline** communicated after assessment
- **Credit** in our security acknowledgments, if desired

### Scope

This policy applies to:

- coindisplayhub.com and all subdomains
- API endpoints at coindisplayhub.com/api/\*
- Smart contracts deployed by COINDISPLAYHUB

### Out of Scope

- Social engineering attacks against staff or users
- Denial of service attacks
- Issues in third-party services or dependencies
- Vulnerabilities requiring physical access to infrastructure

### Safe Harbor

We will not pursue legal action against security researchers who:

- Act in good faith to avoid privacy violations, data destruction, or service disruption
- Report vulnerabilities through the channels described above
- Allow reasonable time for resolution before any public disclosure

## Supported Versions

We support only the latest production release of our platform.
